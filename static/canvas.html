Clients Connected 1
Backend Specs

Engine: Go (Golang)

Concurrency: 12-Shard Hub

Storage: Redis Bitfield (u4)

Protocol: Binary WebSockets

About

Owner: Asendar

This is a collaborative pixel art canvas, inspired by r/place. Place pixels, create art together!

Technical Overview: Built to handle real-time WebSocket communication and high-frequency canvas updates. This project explores concurrent data handling and efficient state synchronization between multiple clients.

Instructions: Select a color from the palette and click on the canvas to place a pixel. Updates from others are instant and in real time!
